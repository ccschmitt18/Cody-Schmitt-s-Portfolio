This was the first map I ever created using ESRI's ArcGIS and Adobe Illustrator. This was a result of the first lab in my introductory cartography course. For this project, we were instructed to select a state and create a point of interest map for that state and create an inset map to show the state's location relative to the others. I chose Nevada, and picked five archaeological cave sites in the state.
