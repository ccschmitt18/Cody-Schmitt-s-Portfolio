This is the result of my third lab in the Graphic Design in Cartography course. In this lab, we were tasked with deconstructing an existing aesthetic style to its constituent form, colors, type, and textures. We took these stylistic components to a multiscale, "slippy" tilesheet using CartoCSS which extended our cartographic design workflow to the web for the first time. Along the way, we created our own set of tiles using Mapbox Studio Classic. I chose the style Stradano used in depicting illustrations of Dante's Inferno. To see my entire slippy map, click here
